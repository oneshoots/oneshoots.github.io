6. what is partitive proportion?[tex]\left \{ {{y=2} \atop {x=2}} \right. \left[\begin{array}{ccc}1&2&3\\4&5&6\\7&8&9\end{array}\right][/tex]


There are a couple of examples that we can break down the 7x9 equation using the distributive property. One is we can expand one of the values in the equation.


For this example, lets expand the equation from 7x9 to 7(4+5). Expanding this new equation further, it would be turned into 7x4 + 7x5. Following the PEMDAS method, it would bring us to 28 + 35, which would produce the sum of 63, which is the same answer for the first equation of 7x9.


You can also do this with the other number, in this case, change it from 7x9 to, say, (2+5)9. Expanding it further will give us 9x2 + 9x5, which will give us 18 + 45, with a sum of 63, same as the previous equations.

24. If there are 9.20 volts across a 4.75 ohm resistor, what is the current? a.1.94 Amp c. 4.45 Amp b.13.95 Amp d. 43.70 Amp


Answer:

Hi :D

Ohm's law states that:

[tex]\boxed{\sf{\blue{Current = \dfrac{Voltage}{Resistance} }}}[/tex]

Voltage = 9.20 V.

Resistance = 4.75 Ω

Substitute.

[tex]\boxed{\sf{\blue{Current = \dfrac{9.20}{4.75} }}}[/tex]

[tex]\boxed{\sf{\blue{Current =1.94 \ A }}}[/tex]

[tex]\frak Current \ is \ 1.94 \ Amperes.[/tex]

[tex]\mathcal{THATS \ IT!}[/tex]
