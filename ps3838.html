18. If I used 185 kilowatt and paid P3136 for the month of September and used 316 kilowatt and paid P5338 for the month of October. What would be my usage and price to pay for the month of November? (pls show solution im begging)


Answer:

Device Wattage (watts) x Hours Used Per Day = Watt-hours (Wh) per Day. Example: A 125-watt television used three hours per day. ...

Device Usage (Wh) / 1000 (Wh/kWh) = Device Usage in kWh. ...

Daily Usage (kWh) x 30 (Days) = Approximate Monthly Usage (kWh/Mont

Step-by-step explanation:

To solve this problem, we need to determine the cost per kilowatt hour by dividing the total cost by the total usage for each month.

For September: P3136 / 185 kilowatts = P16.89 per kilowatt hour

For October: P5338 / 316 kilowatts = P16.87 per kilowatt hour

Now that we have the cost per kilowatt hour for September and October, we can use this information to estimate the cost for November. Let's say that you expect to use 300 kilowatts in November.

Estimated cost for November: 300 kilowatts x P16.89 per kilowatt hour = P5067

This is just an estimate and the actual cost may vary based on factors such as changes in the rate per kilowatt hour or any applicable fees or discounts.
