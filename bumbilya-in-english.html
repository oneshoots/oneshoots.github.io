14. ano po ang mas malakas ang ilaw....bumbilya ohh araw?


Mas malakas ang ilaw ng araw sapagkat ito ang pangunahing pinagmumulan ng ilaw ng buong mundo samantalang ang bumbilya ay isang pansamantalang liwanag lamang.

Sa pagtatanim, mas nakakatulong ang araw kaysa sa ilaw ng bumbiliya. Ang araw ay direktang nagbibigay ng enerhiya samantalang ang bumbiliya ay nangangailangan ng kuryente upang magbigay ng enerhiya. Mas maraming enerhiyang naibibigay ang araw kaysa sa bumbilya. Ayon sa mga research, ang araw ang mas magandang ilaw para sa mga halaman.

17. halimbawa ng bumbilya or light blob​


BUMBILYA NG ILAW_____________________Ano ang bumbilya?

• Ang bumbilya ng ilaw o ilaw na dekuryente ay isang uri ng aparato na lumilikha ng liwanag sa pamamagitan ng paggamit ng kuryente. Ito ay ginagamit kapag madilim ang isang sulok o bahagi ng isang lugar upang makapagbigay ng liwanag.

Halimbawa ng bumbilya:

• Silid o klasikong lampara.

• Navigator

• Bombilya ng halogen.

• Mga LED bombilya.

• Fluorescent tubular light.
