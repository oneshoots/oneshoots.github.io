Hello!

Isotopes are atoms or chemical elements that have the same atomic number (Z), that is, the same number of protons (p), but differ in the number of neutrons (n).

[tex]\boxed{Z =p \neq n}[/tex]

Example: Carbon Isotopes

[tex]\underbrace{C^{12}_6\:\:\:\:C^{13}_6\:\:\:C^{14}_6}_{isotopes}[/tex]

Answer:

a. the same number of protons but different numbers of neutrons.
