1. A three-digit number has the following properties:1. When we change the units digit with the tens digit, it increases by 18 units;2. When we change the tens digit with the hundreds digit, it increases by 180 units.3. How many units will this number increase if we exchange the units digit with the hundreds digit?​


The number will increase by 396 units.

Let's assume that the three-digit number is ABC. Swapping the units digit with the tens digit, we get ACB.

1. We have the information that ABC = ACB + 18. Note that this expression can be written as follows:

100A + 10B + C = 100A + 10C + B + 18

9B - 9C = 18

B - C = 2

B = 2 + C.

2. Now, let's change the tens digit to the hundreds digit. So we have: BAC. Also, ABC = BAC + 180. In the same way as we did before, we get:

100A + 10B + C = 100B + 10A + C + 180

90A - 90B = 180

A - B = 2.

Realize that B = 2 + C. Then:

A - (2 + C) = 2

A - 2 - C = 2

A - C = 4.

3. If we change the units digit with the hundreds digit, the number will be CBA = 100C + 10B + A. Subtracting ABC - CBA, we find:

ABC - CBA = 100A + 10B + C - (100C + 10B + A) = 100A + 10B + C - 100C - 10B - A = 99A - 99C = 99(A - C) = 99.4 = 396.

y be the unit digit
xy - the original number

y = x + 5

x + (x + 5) = 1/2(10x + x + 5) - 8
2x + 5 = 5x + 1/2(x + 5) - 8

x = 3

y = 3 + 5

28. Determine the units digits (one digit) of 4200


Answer:

1. For 4^200, let's first try looking at the pattern of raising 4:

4^1 = 4

4^2 = 16

4^3 = 64

4^4 = 256

See the pattern so far? If the exponent is odd, the units digit is 4 and if the exponent is even, the units digit is 6.

So since the exponent you have is 200, let's try proving that if the exponent is even, the last digit is 6. Let's try using the principle of mathematical induction (PMI). How PMI works is to first test it out using the first number in the pattern. In this case, yes, we've seen that 4^2 = 16, which has 6 as the last digit.

Now, step 2 is to assume that it works for 4^(2k), where k is a natural number.

Step 3: Using the assumption in step 2, we have to show that the last digit of 4^(2(k+1)) is 6.

Well, 4^(2(k+1)) = 4^(2k+2) = 4^(2k) x 4^2 = 4^(2k) x 16.

Since we assumed in step 2 that 4^(2k) ends in 6, we have to multiply (a number ending in 6) x 16.

The ones digit of this is the ones digit of 6x6 = 36, so that's 6. Thus it's correct. We can now be sure that the unit digit is 6.

If that's a bit too much, you can just think of it like 4^200 = 16^100. But multiplying 16 to itself, since the ones digit is always from 6x6 = 36, the unit digit is 6.

For number 2:

Let's let x be the number of ducks, and y be the number of pigs.

We have the following equations:

1. x + y = 35

2. 2x + 4y = 98

The first one from the total animals in the field, and the second, from their legs. We can now use a system of linear equations. To simplify, we can divide the second equation by 2 to get:

3. x + 2y = 49

Subtracting equation 1 from equation 3:

2y - y = 49 - 35

y = 14. So we have 14 pigs.

Substituting this back to equation 1:

x + 14 = 35

x = 21, so 21 ducks.

We can check this by substituting in the 2 equations.

1. 21 + 14 = 35, correct.

2. (2 x 21) + (4 x 14) = 98, correct.
