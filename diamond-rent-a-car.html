To determine the number of mall cars and large cars rented, we can set up a system of equations using the information provided. Let's define the following variables:

x: number of mall cars rented

y: number of large cars rented

We are told that the students rented 4 times as many large cars as mall cars, which means that y = 4x.

We are also told that the cars altogether can hold 56 people. Since each mall car can hold 4 people and each large car can hold 6 people, the total number of people that the cars can hold is 4x + 6y.

We can set up the system of equations as follows:

y = 4x

4x + 6y = 56

We can solve this system of equations to determine the number of mall cars and large cars rented.
