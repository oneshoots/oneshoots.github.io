Answer:

Solving quadratic equation by extracting square roots:

Example:

x² - 16 = 0

Ang una mong gagawin ay ililipat mo yung  - 16 or yung constant sa right side para matira yung x² sa left side.

Parang ganto:

x² = 16

Tandaan na kapag naglilipat tayo ng number sa kabilang side laging nagiiba yung sign nung nilipat na number. Kaya naging 16 yan.

Next ay isqsquare mo silang dalawa para maging x na lang yung x² kase macacancel yung exponent nya.

Ganto:

[tex]\sqrt{x^2}[/tex] = [tex]\sqrt{16}[/tex]

Tandaan na sa extracting square root, lagi natin silang isqsquare para makuha ang value ng x at macancel ang exponent. Lagi rin nating nililipat ang constant number sa kabilang side para mas madali syang iextract.

Pag naextract na naten sila, matatanggal na naten yung exponent ng x tsaka malalaman mo na yung value nya.

x = [tex]\sqrt{16}[/tex]

Para malaman ang value, isolve naten yung [tex]\sqrt{16}[/tex] kase sya lang yung constant at pwede masolve.

Solve:

[tex]\sqrt{16}[/tex] = 4 and - 4

Ang square root ng 16 ay 4 at - 4

So ang x naten will be:

x₁ = 4, x₂ = -4

15. how to solve quadratic equation? Tagalog please HIHI​


Answer:

Ang isang paraan ng paglutas para sa x sa isang quadratic equation ay sa pamamagitan ng pagkumpleto ng parisukat. Narito kung paano:

x ^ 2 + 6x - 7 = 0 ===> palakol ^ 2 + bx + c = 0

==================

1) Ilipat ang pare-pareho sa kanang bahagi. Sa equation sa itaas, ang pare-pareho ay 7. Maaari mo itong gawin sa pamamagitan ng pagdaragdag ng negatibong halaga ng pare-pareho sa magkabilang panig ng equation;

x ^ 2 + 6x -7 + (+7) = 0 + (+7)

x ^ 2 + 6x = 7

2) Kunin ang kalahati ng halaga ng x-term o b (6).

6/2 = 3

3) Parisukat ito.

3 ^ 2 = 9

4) Idagdag ang resulta sa magkabilang panig.

x ^ 2 + 6x + 9 = 7 + 9

5) I-convert ang kaliwang bahagi sa parisukat na form at gawing simple ang kanang bahagi.

(x + 3) ^ 2 = 16

6) Kunin ang parisukat na ugat ng magkabilang panig.

x + 3 = +/- 4

7) Solve para sa x

x + 3 = 4

x = 4 - 3

x = 1

====

x + 3 = -4

x = -4 - 3

x = -7

=========

Mga sagot:

x = 1 at -7
