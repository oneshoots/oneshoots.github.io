4. Paano to? Sagutin diko alam nakakalito


Hi!


Let's first review what a geometric sequence is. It's a sequence where all the terms have a common ratio, i.e., you get the next terms by multiplying the same number to the previous terms. For example:

3 6 12 24 48, ... is a geometric sequence with a common ratio of 2 because 6/3 = 2, 12/6 = 2, 24/12 =2, ...


Now for #1:

The sequence is 3/20, 3/2, 15, ...

Let's first find the common ratio, r, by dividing consecutive terms, so:

[tex] \frac{3}{2} / \frac{3}{20} = \frac{3}{2} * \frac{20} {3} = 10 [/tex]

We can check that 15 / (3/2) = 10 also.

This means r = 10.

So to get the 5th term, we continue to multiply 10 to each term.

3/20, 3/2, 15, 150, 1500

Or, you can use the given formula:

[tex] a_n = a_1r^{n-1}
\\ a_5 = (\frac{3}{20})10^{5-1}
\\a_5 = (\frac{3}{20})(10,000)
\\a_5 = 1500 [/tex]


For #2:

We are asked for [tex]a_6[/tex]. Given are [tex]a_2 = 6, r = 2[/tex].

We know that [tex]6 = a_2 = a_1r = 2a_1[/tex], by substituting r=2.

We can get that [tex] 6 = 2a_1 [/tex], or [tex] a_1 = 3 [/tex].

Now, use the given formula to get

[tex] a_6 = a_1r^{6-1}

\\a_6 = 3*(2)^5

\\a_6 = 3*32 = 96[/tex].


Okay, now for #3:

For k-3, k+1, and 4k-2 to form a geometric sequence, they must have a common ratio. This means that [tex] (k-3)r = k+1 [/tex], and [tex] (k+1)r = 4k-2 [/tex].


Expanding both, we get:

[tex]kr -3r = k + 1

\\kr + r = 4k - 2[/tex].


We can subtract the two equations to eliminate kr, this gets us:

[tex] -4r = -3k +3[/tex]


Isolating r,

[tex] r = \frac{-3k+3}{-4} = \frac{3k - 3}{4}[/tex].


Substituting this into [tex] (k-3)r = k+1 [/tex], we get:

[tex] \frac{(k-3)(3k-3)}{4} = k+1

\\ 3(k-3)(k-1) = 4(k+1)

\\ 3(k^2 -4k +3) = 4(k+1)

\\ 3k^2 - 12k + 9 - 4k - 4 = 0

\\ 3k^2 -16k + 5 = 0

\\ (3k -1)(k-5) =0

\\ k = \frac{1}[3] , 5 [/tex].


To be sure let's check both possible solutions by substituting them into the original problem: k-3, k+1, and 4k-2


If k = 1/3, the sequence is

-8/3 , 4/3, -2/3

We can see that r = -1/2, so k=1/3 works.


If k =5, the sequence is

2, 6, 18

Here, r=3, so k =5 is also correct.


Hay! #4 , almost there:

Given sequence 6, 12, 24, 48, ... , which term is 768?

From the formula [tex] a_n = a_1r^{n-1}[/tex]

We are given [tex]a_1 = 6, a_n = 768[/tex]. We can also get that r= 12/6 = 2.

The question is, what is n?

By just plugging everything in, we get:

[tex] 768 = 6*2^{n-1}

\\ 128 = 2^{n-1} [/tex]


If you're familiar with the powers of 2, (2,4,8,16,32,64,128,256, ...), you can see that n-1 = 7, so n = 8. Or, we can use logarithms:

[tex] log 128 = (n-1) log 2

\\ log 128 / log 2 = n - 1

\\ 7 = n-1

\\ n=8

[/tex]


Finally, #5:

We are asked to find [tex]a_1[/tex], given that [tex]a_2 = \frac{3}{4}, a_4=3 [/tex].


Using the formula again,

[tex] a_2 = a_1r^{2-1} = a_1r = \frac{3}{4}

\\ a_4 = a_1r^{4-1} = a_1r^3 = 3[/tex].


Substituting the 1st equation into the second,

[tex] 3 = a_1r^3 = (a_1*r)r^2 = \frac{3}{4}r^2

\\ 3 = \frac{3}{4}r^2

\\ 3*\frac{4}{3} = r^2

\\ 4 = r^2

\\ r = 2, -2

[/tex]


We can now use r to get [tex] a_1 [/tex].

From [tex] a_2 = a_1r = \frac{3}{4}

\\ a_1 = \frac{3}{4r}

\\ a_1 = \frac{3}{8}, \frac{-3}{8}

[/tex]


Checking:

If r=2,

3/8, 3/4, 3/2, 3


If r= -2

-3/8, 3/4, -3/2, 3


Both work.
